鲸落 - JavaScript学习笔记

JavaScript

欢迎来到我的博客。本部分主要是记录我学习JavaScript笔记以及在学习过程中遇到的一些问题。

1.基础笔记

  • js数据类型

    • 基本数据类型：字符串(String)、数值(Number)、布尔值(Boolean)、未定义(Undefined)
    • 除基本数据类型，其他都是对象(Object)

  • js几种输出方式

    • 直接输出：document.write()
    • 弹出提示框： alert()
    • 打印到控制台：console.log()
    • 弹出输入框：prompt()

  • typeof：检查数据类型，eg:var a;console.log(typeof a);,打印结果：number

  • 逻辑运算符

    • !(非)
    • &&(与)：true&&true为true、true&&false为false、false&&true为false、false&&false为false
    • ||(或)：true||true为true、true||false为true、false||true为true、false||false为false

  • js中对象的分类

    • 内建对象：像String、number、function等
    • 宿主对象：由浏览器提供，如DOM、BOM等
    • 自定义对象
  • a+=5与a=a+5等效，同理+、-、*、/、%也是
  • =表示赋值，==表示判断(相同返回true，不同返回flase)，===表示绝对相等，即数据类型与值都必须相等

2.区别解析

  • a++与++a的区别
  • ES6中var let constant 区别
  • break、continue、return的区别
    • break：退出当前循环
    • continue：跳过当次循环
    • return：终止函数的执行并返回函数的值

3.

  • ES6中箭头函数的this指向
  • js中定义var that = this 与 var _this = this 的作用
